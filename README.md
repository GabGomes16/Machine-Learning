# Machine Learning – UFMG (2024/2)

This repository was created to store the assignments and practical projects developed during the **Machine Learning** course at UFMG (2024/2 semester).

The course covered the statistical foundations of machine learning, data analysis techniques, and predictive modeling methods.

---

## 📚 Course Topics

### 📊 Statistical Foundations
- Statistical principles of Machine Learning  
- Correlation methods  
- Causal relationships  

### 🔎 Exploratory Data Analysis
- Exploratory Data Analysis (EDA)  
- Data visualization techniques  

### 📈 Regression Models
- Linear Regression  
- Logistic Regression  

### 🧪 Model Evaluation
- Data partitioning (train/test split)  
- Resampling methods  

### 🌳 Machine Learning Methods
- Clustering techniques  
- Tree-based classifiers and regressors  

---

## 💻 Tools & Languages

Projects were developed using:
- Python
- Jupyter Notebook
- Statistical and Machine Learning libraries

This repository reflects hands-on experience with statistical modeling, data exploration, and predictive algorithms.
